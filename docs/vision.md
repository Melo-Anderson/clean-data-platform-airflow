# Plataforma de Dados

## Problema

Construir uma plataforma capaz de orquestrar:

- ingestão
- transformação
- integração
- descoberta de metadados

para diferentes tipos de fontes de dados.

## Objetivos

- Reduzir desenvolvimento manual
- Facilitar onboarding de novas fontes
- Padronizar ingestões
- Permitir observabilidade

## Fora de Escopo

- BI
- Machine Learning
- Catálogo corporativo
