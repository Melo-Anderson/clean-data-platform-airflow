# Core Platform CI/CD Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Set up a GitHub Actions CI pipeline to run unit tests and build the Docker image for the Core Data Platform repository.

**Architecture:** A standard GitHub Actions workflow triggered on push and pull requests to `main`. It will use `astral-sh/setup-uv` to quickly install dependencies, run `pytest`, and finally test the Docker image build.

**Tech Stack:** GitHub Actions, `uv`, Docker, Pytest.

## Global Constraints

- Must use `uv` for python dependency installation.
- Must run the existing `pytest` test suite.
- Must verify that `Dockerfile.api` builds successfully.

---

### Task 1: Create GitHub Actions Workflow

**Files:**
- Create: `.github/workflows/ci.yml`

**Interfaces:**
- Consumes: `pyproject.toml`, `Dockerfile.api`, `tests/`
- Produces: GitHub Actions CI workflow file.

- [ ] **Step 1: Write the GitHub Actions workflow YAML**

```yaml
name: Core Platform CI

on:
  push:
    branches:
      - main
      - develop
  pull_request:
    branches:
      - main
      - develop

jobs:
  test:
    name: Run Unit Tests
    runs-on: ubuntu-latest
    steps:
      - name: Checkout Code
        uses: actions/checkout@v4

      - name: Setup uv
        uses: astral-sh/setup-uv@v3
        with:
          enable-cache: true

      - name: Set up Python
        uses: actions/setup-python@v5
        with:
          python-version: "3.12"

      - name: Install Dependencies
        run: uv sync --no-dev
        
      - name: Install Test Dependencies
        run: uv sync

      - name: Run Tests
        run: uv run pytest tests/ -v

  build:
    name: Build Docker API Image
    runs-on: ubuntu-latest
    needs: test
    steps:
      - name: Checkout Code
        uses: actions/checkout@v4

      - name: Build Docker Image
        run: docker build -t platform-api:latest -f Dockerfile.api .
```

- [ ] **Step 2: Commit the workflow file**

```bash
git add .github/workflows/ci.yml
git commit -m "ci: add github actions workflow for tests and docker build"
```
